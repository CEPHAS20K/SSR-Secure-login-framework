# SSR-Secure-login-framework

this is a full pack ssr login and admin dashboard crete using pug template engine and express.js it comes with production ready features lots of them

# Auth Project

Quick start from project root:

```bash
npm run setup
npm run dev
```

Main commands:

- `npm run setup`: install root, backend, and frontend dependencies.
- `npm run dev`: run backend dev server and frontend Tailwind watcher together.
- `npm run dev:backend`: run backend only.
- `npm run dev:frontend`: run frontend watcher only.
- `npm run build`: build frontend CSS.
- `npm run check`: run formatting checks and frontend build.
- `npm run start`: start backend with development env.
- `npm run start:proc`: build frontend and start backend with production env.
